## Install 

```
$ npm install --global page-id-of
```
__OR__
```
$ sudo npm install --global page-id-of
```

## Usage 

```
$ page-id-of

	Usage   : page-id-of <page's name>

	Example : page-id-of Humorous.Sperm.Official

	Help    : page-id-of -h --help
```
